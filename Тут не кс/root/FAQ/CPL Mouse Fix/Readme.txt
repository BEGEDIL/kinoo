off_acceleration.reg — отключает акселерацию мыши в CS 1.6 и во всей ОС
on_acceleration.reg — восстанавливает стандартные настройки реестра, что включает акселерацию мыши в CS 1.6 и во всей ОС

1) Откройте файл двойным щелчком, на вопрос "Вы действительно хотите добавить информацию из X:\..\off_acceleration.reg?" ответьте "Да"
2) Перезагрузите компьютер для применения изменений
